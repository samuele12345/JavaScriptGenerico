# JavaScriptGenerico

Questa cartella principale raccoglie una serie di esercizi generici relativi a JavaScript, HTML e CSS.

L'obiettivo del repository e mostrare esempi semplici e progressivi su argomenti come:

- gestione di input utente
- manipolazione del DOM
- eventi e addEventListener
- piccoli componenti interattivi
- uso base di Math e conversioni di tipo
- stile e layout con CSS
- metodi utili per lavorare con le stringhe

Ogni sottocartella contiene un esercizio separato, pensato per provare un concetto specifico in modo pratico.

Struttura generale della repository:

- conterProgram: contatore con incremento, decremento e reset
- ottenereInput: lettura e validazione di un input testuale
- random: generazione di numeri casuali tra minimo e massimo
- payment: selezione di un metodo di pagamento con interfaccia semplice
- strings: esempi pratici dei principali metodi per le stringhe
- math: esempi delle principali funzioni dell'oggetto Math
- conversioneLet: esempi di conversione tra tipi primitivi
